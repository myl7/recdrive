# recdrive

中科大睿客网云盘 rec.ustc.edu.cn/recdrive (unofficial) SDK

## Features

- List a folder
- Upload / Download a file

## Get Started

```bash
# List a folder
recdrive -token <token> ls <dir>
# Upload a file
recdrive -token <token> cp <file> :<dir>
# Download a file
recdrive -token <token> cp :<file> <file>
```
